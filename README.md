## About The Project

Recently, my personal trainer in my gym made a nutrition plan for me to be more healthy, however the problem is I really don't want to give up my habit hanging out with friends in restaurants.

As the key point is to maintain a calories, I create this project to track my calories intake with the constrains in protein, carb, and fat...

## Usage
First, you need to choose the restaurant that you want to order

On the left panel, choose your goals in take, the algorithm will run and automatically tell you the calories that you are going to consume and which food you should order
